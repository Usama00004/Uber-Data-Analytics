# Uber Data Analytics 

## Project Overview

The **Uber Data Analytics Dashboard** is an interactive business intelligence report built using **Microsoft Power BI** to analyze Uber ride data.  

This dashboard transforms raw trip data into meaningful insights that help understand ride patterns, revenue performance, peak demand hours, and location-based trends.

The objective of this project is to demonstrate data cleaning, modeling, DAX calculations, and interactive visualization skills using real-world transportation data.

-

##  Dashboard Features

### 1️ Trip Analysis
- Total Trips
- Trips by Date, Month, and Year
- Peak Hours Identification
- Day-wise and Weekday Trends

### 2️ Revenue Insights
- Total Revenue
- Average Fare per Trip
- Revenue Trends Over Time
- Revenue by Ride Category

### 3️ Location Analysis
- Pickup & Drop-off Hotspots
- Most Frequent Routes
- Location-wise Demand Distribution

### 4️ Ride Metrics
- Average Trip Distance
- Trip Duration Analysis
- Ride Category Distribution

### 5️ Interactive Capabilities
- Date Filters (Slicers)
- Location Filters
- Ride Type Filters
- Drill-down Reports

-

##  Tools & Technologies Used

- **Microsoft Power BI** – Data visualization & dashboard development  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Custom measures and KPIs  
- **Excel / CSV Dataset** – Uber trip data  

-

##  Dataset Description

The dataset includes the following fields:

- Trip ID  
- Pickup Date & Time  
- Drop-off Date & Time  
- Pickup Location  
- Drop-off Location  
- Trip Distance  
- Fare Amount  
- Payment Type  
- Ride Category  

-

##  Key Business Insights

- Identified peak demand hours and busiest days
- Discovered high-revenue locations
- Analyzed correlation between trip distance and fare amount
- Evaluated performance of different ride categories
- Tracked revenue growth trends over time
